\subsection{\-Detailed \-Description}
\-Specialist tree widget class consiting of \-G\-C\-Tree\-Widget\-Items. 

\-This class wraps an underlying \-D\-O\-M document and manages its items (\-G\-C\-Tree\-Widget\-Items) based on changes to the \-D\-O\-M, but also manages the \-D\-O\-M based on changes made to its items. \-Perhaps a better way of describing this relationship is to say that this class is, in effect, a non-\/textual visual representation of a \-D\-O\-M document.
