\subsection{\-Detailed \-Description}
\-Specialist text edit class for displaying \-X\-M\-L content in the \-X\-M\-L \-Mill context. 

\-Provides functionality with which to comment out or uncomment \-X\-M\-L selections and keeps track of which \-X\-M\-L nodes are currently under investigation (based on cursor positions).
